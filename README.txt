SIGI — Sistema Integrado de Gestão para Igrejas
Versão 2.0 do protótipo funcional acadêmico.

Arquivos:
- index.html
- style.css
- script.js

Como publicar no GitHub Pages:
1. Abra o repositório SIGI no GitHub.
2. Clique em Add file > Upload files.
3. Envie estes três arquivos substituindo os antigos.
4. Clique em Commit changes.
5. Aguarde alguns minutos e acesse:
   https://gabvedoy-dev.github.io/SIGI/

Observação:
Este projeto é um protótipo navegável em HTML, CSS e JavaScript.
Não possui banco de dados, login real ou armazenamento de documentos.
